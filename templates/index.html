Hello, world!

This is a simple interface that aims to show the results of diverse object detectors trained on PASCAL VOC 07+12. The purpose of this project is to concretely compare the results of the Few-shot learning & transfer learning against the results by state-of-the-art detectors.

The detectors available are :

  • FSOD - Feature reweighting
  • Vanilla - MobileNet
  • FSOD - RepMet
